# sunvoy-challenge

## Setup

1. `npm i`

1. Copy `credentials.sample.json` to `credentials.json` and modify to real credentials (which are not in this github repo.)

1. `npm run start`

1. Remove stored login data (cookies) by running `npm run clean`.
